POS User Access Rights

You are able to manage and supervise POS User Access Rights with this module.



Features
  • POS - Enable/Disable Discount.
  • POS - Enable/Disable Payment
  • POS - Enable/Disable Numpad.
  • POS - Enable/Disable Refund.
  • POS - Enable/Disable Plus-Minus.
  • POS - Enable/Disable Customer.
  • POS - Enable/Disable Qty.
  • POS - Enable/Disable New Order.
  • POS - Enable/Disable Remove Order Line.
  • POS - Enable/Disable Delete Order.
  • POS - Enable/Disable Price.
  • Support Community and Enterprise Editions
All of the pos access functions are available from the user under the pos access tab.

Clicking on the feature from the user pos access tab will enable it.
