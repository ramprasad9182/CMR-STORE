ALL in one POS Reports(POS BOX Compatible) Odoo Apps

Using this Odoo apps you can print many types of reports or receipts like, Sales Summary Report, Ongoing Session Report, Posted Session report from backend and Payment Summary, Category Summary, Product Summary, Order Summary, Session Audit Report, Inventory Audit Report from Point of Sales Screen in Odoo and This odoo apps is POS-BOX Compatible.
