HSN Code Automation Management

Facilitates the generation of HSN codes with corresponding GST (tax) rates, tailored to the company's specified category. These codes can then be seamlessly applied to products, enabling the creation of sales orders, purchase orders, invoices, and bills. Additionally, taxes (GST) are automatically applied based on the selected product, with further adjustments made according to the applicable state regulations.

Pre-Installation Requirements

@ pip install bs4
